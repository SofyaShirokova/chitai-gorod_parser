# Chitai-Gorod Parser

Парсер для сбора данных о комиксах с сайта Читай-город. Скрипт извлекает информацию о названии, авторе, рейтинге и ценах товаров из категории комиксов.

## Установка

Для запуска проекта необходимо:

- Python 3.7 или выше
- pip (менеджер пакетов Python)

Установка зависимостей:

    pip install requests beautifulsoup4 pandas

## Использование

Запуск парсера:

    python parser.py

После выполнения скрипта данные сохраняются в файл `comics_data.txt` в той же директории.

## Что парсится

Для каждого товара извлекается:
- Название комикса
- Автор (если указан)
- Рейтинг (если доступен)
- Старая цена (при наличии скидки)
- Актуальная цена

## Формат выходных данных

Данные сохраняются в текстовом файле в формате:

    Название: [Название комикса]
    Автор: [Имя автора]
    Рейтинг: [Значение рейтинга]
    Старая цена: [Цена без скидки]
    Новая цена: [Текущая цена]
